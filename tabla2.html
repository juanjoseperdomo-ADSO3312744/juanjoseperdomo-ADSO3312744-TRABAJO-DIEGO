¿Para qué sirve el < colgroup? >

Un colgroup sirve para poder colocar el diseño a las diferentes columnas de una lista, gracias a este se puede modificar todo, el anchor, el color, el largor, etc. nos sirve para definir columna por columna en una tabla mas rapido y facil, además de que es muy rápido y eficiente.

¿Para qué sirven los < th> ?

Los < th> nos sirven para definir celdas de encabezado en una tabla, este código permite poner títulos o descripciones que nos ayuden a identificar lo que va a contener las diferentes tablas.
